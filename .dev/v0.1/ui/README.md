## 设计要点

### 布局结构
```
Header (固定)
├── Logo + 导航 + 用户信息

Main Layout
├── Sidebar (280px 固定宽度)
│   ├── 板块来源下拉框
│   ├── 新建板块按钮
│   ├── 板块个股列表
│   └── 所属板块区域（搜索后显示）
│
└── Main Content (自适应宽度)
    ├── 工具条（搜索框 + 类型切换）
    └── K 线图区域
```

### 核心交互

1. **板块切换**
   - 通过顶部下拉框切换板块来源（自定义/tushare）
   - 加载对应板块列表
   - 默认选中第一个板块，显示其个股

2. **个股选择**
   - 点击左侧个股列表，右侧显示 K 线
   - 第一个个股默认选中

3. **搜索功能**
   - 顶部工具条支持模糊搜索个股/指数
   - 搜索后显示"所属板块"区域
   - 点击所属板块可快速切换

4. **新建板块**
   - 点击按钮弹出模态框
   - 支持文件上传或手动输入

## 使用方式
