## 完成内容

### 系统设置需求文档

新建 `.dev/specs/04-system_settings.md`，内容包括：

1. **定时任务** (`/system/jobs`)
   - 预置 5 个任务（不可新增/删除）
   - 任务列表 + 启用/禁用操作
   - 展开详情显示最近 5 次执行记录
   - 手动触发执行
   - API: `/api/system/jobs` 等

2. **交易网关** (`/system/gateway`)
   - 显示当前配置（不暴露密钥明文）
   - 连接状态实时显示
   - 连接测试功能
   - 提示重新配置走初始化向导
   - API: `/api/system/gateway/status`, `/api/system/gateway/test`

3. **数据源** (`/system/datasource`)
   - 显示 Tushare Token、数据起始日、数据覆盖范围
   - 各数据类型完整性检查状态
   - 最近同步记录
   - 手动触发全量同步（带进度）
   - API: `/api/system/datasource/status`, `/api/system/datasource/sync`

通用说明：
- 页面样式与数据维护模块一致
- 仅 admin 用户可访问
