# 系统设置

本模块提供系统运行时配置、定时任务管理、数据源管理等功能。

## 定时任务

### 功能概述

提供定时任务的可视化管理，包括查看任务列表、启用/禁用任务、手动触发执行、查看最近执行结果。

### 预置任务

系统预置以下任务，不允许用户删除或新增：

| 任务 ID           | 任务名称     | 调度时间     | 说明                         |
| ----------------- | ------------ | ------------ | ---------------------------- |
| `daily_bars_sync` | 日线数据同步 | 每天 15:35   | 从数据源下载当交易日行情数据 |
| `stock_list_sync` | 股票列表同步 | 每天 16:00   | 更新证券列表（上市/退市）    |
| `calendar_sync`   | 交易日历同步 | 每周一 09:00 | 更新节假日和调休数据         |
| `daily_snapshot`  | 日终快照     | 每天 15:05   | 记录当天收盘持仓快照         |
| `market_snapshot` | 行情快照     | 每 5 分钟    | 记录最新行情数据             |

### 任务列表

界面示意如下：

```
┌─────────────────────────────────────────────────────────────────────┐
│ ⚙ 系统设置 / 定时任务                                                │
├─────────────────────────────────────────────────────────────────────┤
│  任务名称              调度时间        状态      上次执行   操作     │
│ ─────────────────────────────────────────────────────────────────── │
│  日线数据同步          每天 15:35     🟢 运行中  成功      [禁用]   │
│  股票列表同步          每天 16:00     🔴 已停止  --        [启用]   │
│  交易日历同步          每周一 09:00   🟢 运行中  失败      [禁用]   │
│  日终快照              每天 15:05     🟢 运行中  成功      [禁用]   │
│  行情快照              每 5 分钟      🟢 运行中  成功      [禁用]   │
└─────────────────────────────────────────────────────────────────────┘
```

在界面元素中，注意到以下各项：

1. **状态指示器**：运行中（绿色圆点）、已停止（红色圆点）。
2. **操作列**：根据状态显示"启用"或"禁用"按钮，点击后切换状态。
3. **上次执行**：显示最近一次执行结果，成功时显示绿色"成功"，失败时显示红色"失败"并显示错误摘要。
4. **手动触发**：点击任务行可展开详情面板，显示最近 5 次执行记录，并提供"立即执行"按钮。

### API 接口

| 接口                            | 方法 | 说明                  |
| ------------------------------- | ---- | --------------------- |
| `/api/system/jobs`              | GET  | 获取所有任务状态列表  |
| `/api/system/jobs/{id}/toggle`  | POST | 启用/禁用任务         |
| `/api/system/jobs/{id}/run`     | POST | 手动触发一次执行      |
| `/api/system/jobs/{id}/history` | GET  | 获取最近 N 次执行记录 |

## 交易网关

### 功能概述

显示当前交易网关的配置状态，提供连接测试和修改功能。

> 注意：交易网关的完整配置在初始化向导（Step 4）中完成。本页面仅用于查看当前配置和进行连接状态诊断。

### 界面布局

```
┌─────────────────────────────────────────────────────────────────────┐
│ ⚙ 系统设置 / 交易网关                                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  连接状态                                                             │
│  ┌─────────────────────────────────────────────────────────────────┐│
│  │  [🔴/🟢]  gateway.zillionare.com:8000                             ││
│  │                                                                   ││
│  │  状态: 已连接 (延迟: 23ms)                                        ││
│  │  最后同步: 2026-04-10 15:35:02                                   ││
│  └─────────────────────────────────────────────────────────────────┘│
│                                                                       │
│  连接配置                                                             │
│  ┌─────────────────────────────────────────────────────────────────┐│
│  │  服务器地址: localhost                        [连接测试]           ││
│  │  端口:      8000                                                   ││
│  │  API Key:   ••••••••••••                    [显示/隐藏]          ││
│  │  路径前缘:  /                                                     ││

│  └─────────────────────────────────────────────────────────────────┘│
│                                                                       │
│  ⚠️ 如需修改配置，建议重新运行初始化向导。                           │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### 交互说明

1. **连接测试**：点击"连接测试"按钮，向 gateway 的 `/ping` 端点发送请求，显示连接结果（成功/失败及错误信息）。
2. **连接状态**：实时显示 gateway 连接状态。如未启用 gateway，显示"网关未启用"并给出提示。
3. **显示/隐藏 API Key**：点击切换 API Key 的明文/掩码显示。
4. **重新配置提示**：如配置项有变更，提示用户通过初始化向导重新配置，或保存后重启服务使配置生效。

### API 接口

| 接口                         | 方法 | 说明                       |
| ---------------------------- | ---- | -------------------------- |
| `/api/system/gateway/status` | GET  | 获取网关连接状态           |
| `/api/system/gateway/test`   | POST | 测试网关连接               |
| `/api/system/gateway/config` | GET  | 获取网关配置（不包含密钥） |

### 交互说明

1. **数据状态**：实时显示各数据类型的完整性检查结果。如数据缺失严重，状态显示为"⚠️ 数据不完整"并给出缺失时间范围。
2. **手动同步**：点击"手动同步全部数据"，启动数据下载流程，显示进度条和已下载记录数。同步完成后刷新数据状态。
3. **显示/隐藏 Token**：点击切换 Token 的明文/掩码显示。
4. **修改 Token**：如用户修改了 Token，提供保存按钮，保存后重新测试连接有效性。

### API 接口

| 接口                                   | 方法 | 说明                           |
| -------------------------------------- | ---- | ------------------------------ |
| `/api/system/datasource/status`        | GET  | 获取各数据类型的数据状态       |
| `/api/system/datasource/sync`          | POST | 触发全量数据同步               |
| `/api/system/datasource/sync/progress` | GET  | 获取同步进度（SSE）            |
| `/api/system/datasource/config`        | GET  | 获取数据源配置（不包含 Token） |
| `/api/system/datasource/config`        | PUT  | 更新数据源 Token               |

## 通用说明

### 页面样式

所有系统设置页面使用统一的布局风格，与"数据维护"模块保持一致：

- 页面顶部：图标 + 标题（`UkIcon` + `H2`）
- 底部说明文字使用 `text-sm text-gray-500` 灰色样式
- 卡片使用 `bg-white rounded-lg shadow` 样式
- 主内容区 padding 为 `p-8`

### 侧边栏菜单

```
- 数据维护:
    - 交易日历
    - 股票列表
    - 行情数据
- 系统设置:
    - 定时任务
    - 交易网关
    - 数据源
```
