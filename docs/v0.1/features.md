## 功能

本版本提供以下功能：

1. 订阅 xtquant 的全推数据，合成为1m, 30m 和日线，通过 redis 消息队列发布
2. 提供交易账户资金、资产、委托、成交、持仓历史记录，并支持通过 REST API 进行查询
3. 通过 REST提供交易接口，如委买、委卖、撤单等
